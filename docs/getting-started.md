# Getting Started
Jump straight into it by downloading it via one of the recommended ways! Then, learn how to use it using the guides. If you're new, take a look at the intro tutorial, or if you're more technical and ready, take a look at the Seam documentation.

:::warning

As Seam evolves, expect big changes! This means core features and utilities can be changed, added, or removed at any time.

Seam is not yet a 1.0.0+ library since there is no guarantee that bugs won't be present. While it is battle-tested, there can always be an edge-case issue you run into.

If you ever run into any problems, or feel there are features missing, feel free to post an issue on GitHub.

:::

## Installation via .rbxm
Download a current or previous version of Seam via the rbxm (Roblox Studio) format by going here: https://github.com/MiaGobble/Seam/releases/

## Installation via Wally
You can use `wally install` in the terminal to insert this framework into your game. See more here: https://wally.run/package/miagobble/seam

## Installation via source code
The source code for Seam can be found here: https://github.com/MiaGobble/Seam/

## Installation via Roblox Store
You can get it on the Roblox store here: https://create.roblox.com/store/asset/111047414707895/
