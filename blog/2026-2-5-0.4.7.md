v0.4.7 improves scopes and springs, including fixes and features.

<!-- truncate -->

The following was done this update:
* Fixed an issue where setting `Spring.Velocity` did nothing
* `OnEvent` is now scopable
* `OnAttributeChanged` is now scopable
* Updated scope typechecking to fix missing members
* Adjusted Moonwave comments in source code

Also, you may noticed I missed a post about v0.4.6. That update was important for fixing critical issues.

Make sure to keep updating Seam to get the latest features, fixes, and more.
